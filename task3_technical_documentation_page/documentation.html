Variables and Data Types

Python variables do not need explicit declaration to reserve memory space.

Data types are automatically assigned at runtime.

Common types include integers, floats, strings, lists, tuples, and dictionaries.

x = 42 name = "Alice"

Python is dynamically typed.

We can assign any type to a variable at any point in the code.

  • Integer: 1, 2, 3
  • Float: 3.14, 2.71
  • String: "hello"
  • Boolean: True, False
  • Collection: list, tuple, dict, set
Control Flow

Python provides conditional statements like if, elif, and else.

It also provides loops like for and while loops.

if x > 10: print("x is greater than 10")

Break and continue help control the flow within loops.

Exception handling is done using try, except blocks.

  • if-elif-else structure
  • for loops for iteration
  • while loops for conditions
  • break and continue in loops
  • try-except for error handling
Functions

Functions are defined using the def keyword in Python.

They help break the program into smaller, modular chunks.

Python supports default arguments and keyword arguments.

def greet(name="Guest"): print(f"Hello, {name}!")

Anonymous functions can be created using lambda expressions.

  • defining functions
  • default parameters
  • keyword arguments
  • returning values
  • lambda functions
Modules and Packages

Modules are Python files containing Python code such as functions and classes.

Packages are collections of modules organized in directories containing a special __init__.py file.

import math print(math.sqrt(25))

External modules can be installed using pip.

Standard libraries come pre-installed with Python.

  • standard modules
  • third-party packages
  • pip installer
  • virtual environments
  • import statements
